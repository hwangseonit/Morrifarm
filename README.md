## 🚀 Giới thiệu

**Thiết kế Web BTL** là một dự án web dành cho môn học Thiết kế Web, hướng đến việc xây dựng một website hiện đại, thân thiện với người dùng và chuẩn responsive. Website này không chỉ phục vụ mục đích học tập mà còn là nơi thực hành, vận dụng kiến thức về HTML, CSS, JavaScript vào một sản phẩm thực tế.

Sản phẩm mang tính trực quan cao, giao diện sinh động, tốc độ tải trang tối ưu, dễ dàng mở rộng và cập nhật. Dự án cũng chú trọng vào trải nghiệm người dùng (UX/UI), đảm bảo website vận hành tốt trên nhiều thiết bị khác nhau (máy tính, máy tính bảng, điện thoại).

## 🛠️ Công nghệ sử dụng

- 🟧 **HTML5**: Xây dựng cấu trúc nội dung, đảm bảo semantic và chuẩn SEO.
- 🟦 **CSS3**: Thiết kế giao diện, responsive layout, sử dụng flexbox, grid, hiệu ứng chuyển động mượt mà.
- 🟨 **JavaScript**: Tạo tương tác, hiệu ứng động, xử lý sự kiện, kiểm tra đầu vào, v.v.

Ngoài ra, dự án có thể sử dụng các thư viện hoặc framework nhẹ để tối ưu trải nghiệm, nhưng vẫn giữ tính thuần frontend, dễ học và dễ hiểu cho sinh viên.

## 📦 Hướng dẫn sử dụng

### 1. Clone dự án về máy:

```bash
git clone https://github.com/hoaingoc1706/web-design.git
```

### 2. Mở website:

- Chỉ cần mở file `index.html` bằng trình duyệt (Chrome, Firefox, Edge, ...).
- Không cần cài đặt thêm phần mềm khác.

---

## 🗂️ Cấu trúc thư mục

```
web-design/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
│   └── ...
└── README.md
```

- `index.html` : Trang chủ website.
- `css/` : Chứa các file CSS, định dạng giao diện, hiệu ứng.
- `js/` : Chứa các file JavaScript, xử lý logic và tương tác.
- `images/` : Hình ảnh sử dụng trong website.
- `README.md` : Tài liệu hướng dẫn.
